## CSS Classes

The library provides these CSS classes:

| Class | Applied To |
|-------|-----------|
| `input-terminal-stdout` | `<span>` elements for  each `stdout` log |
| `input-terminal-stderr` | `<span>` elements for  each `stderr` log |

### Example Styles

```css
/* stdout styling */
.input-terminal-stdout {
  color: #d4d4d4;
}

/* stderr styling */
.input-terminal-stderr {
  color: #f14c4c;
}
```
